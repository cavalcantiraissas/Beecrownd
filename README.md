# Beecrownd

Repositório com soluções de desafios algorítmicos da plataforma [Beecrowd](https://www.beecrowd.com.br/) (antiga URI Online Judge). O objetivo é manter uma prática contínua de lógica de programação e raciocínio computacional em Python.

Os exercícios cobrem desde problemas introdutórios de entrada e saída até desafios de estruturas de dados e algoritmos de complexidade intermediária.

---

## Organização

Os arquivos estão organizados dentro da pasta `Python/`, nomeados pelo código do problema na plataforma para facilitar a consulta direta.

```
Beecrownd/
  Python/
    1001.py
    1002.py
    ...
```

## Como Usar

```bash
git clone https://github.com/cavalcantiraissas/Beecrownd.git
cd Beecrownd/Python
python 1001.py
```

Para testar com entradas específicas:

```bash
echo "10 20" | python 1001.py
```
